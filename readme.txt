A teaching plugin demonstrating Hooks, Shortcodes, and Core Icons.

== Description ==

WP Demo Lab is a developer-focused teaching plugin designed to demonstrate core WordPress development concepts.

Features included:
*   **Hooks Demo**: Interactive toggles to see `the_content` filter, `wp_footer` action, and `admin_notices` action in real-time.
*   **Shortcodes Demo**: Ready-to-use shortcodes for buttons, dates, and user info.
*   **Icons Gallery**: A searchable,paginated gallery of standard WordPress Dashicons.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-demo-lab` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->WP Demo Lab screen to configure the plugin.

== Frequently Asked Questions ==

= Who is this plugin for? =

This plugin is for developers learning WordPress plugin development or users who want to see how hooks work behind the scenes.
